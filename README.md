# react-native-floating-action-bubble

Floating action bubble

## Installation


```sh
npm install react-native-floating-action-bubble
```


## Usage


```js
import { FloatingActionBubbleView } from "react-native-floating-action-bubble";

// ...

<FloatingActionBubbleView color="tomato" />
```
